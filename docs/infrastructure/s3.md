# S3 Buckets

Documentation for S3 bucket configuration.

## Backup Bucket

| Property | Value |
|----------|-------|
| Name | `wiz-exercise-backups-<random>` |
| Access | **PUBLIC** (intentional vulnerability) |
| Encryption | None |
| Versioning | Disabled |

## Vulnerability: Public Access

This bucket is intentionally configured with public read access. See [WIZ-001: Public S3 Bucket](../security/s3-public.md).

## Terraform Configuration

```hcl
resource "aws_s3_bucket" "backup" {
  bucket = "wiz-exercise-backups-${random_id.suffix.hex}"
}

resource "aws_s3_bucket_public_access_block" "backup" {
  bucket = aws_s3_bucket.backup.id

  block_public_acls       = false  # VULNERABLE
  block_public_policy     = false  # VULNERABLE
  ignore_public_acls      = false  # VULNERABLE
  restrict_public_buckets = false  # VULNERABLE
}
```
